# Contributing to GenAI Mastery Roadmap

Thank you for helping keep this curriculum free, accurate, and up-to-date! Here's how to contribute.

## Types of Contributions

### 🔗 Resource Updates
If you find a better resource for a week (newer video, more accurate blog post, updated docs):
1. Open an issue with the label `resource-update`
2. Include: Week number, current resource URL, proposed resource URL, and why it's better
3. We review and merge within 48 hours

### 🐛 Bug Reports
Found an outdated link, incorrect information, or a broken resource?
1. Open an issue with the label `bug`
2. Include: Week number, the incorrect content, and the correct information

### 📝 New Content
Want to add a new project idea, challenge variation, or supplementary resource?
1. Fork the repo
2. Create a branch: `feature/week-XX-improvement`
3. Make your changes
4. Submit a PR with a clear description

### 🌍 Translations
Want to translate the curriculum?
1. Create a folder: `translations/[language-code]/`
2. Copy and translate `README.md`
3. Submit a PR

## Content Standards

**Resources must be:**
- Free to access (no paywalls)
- Less than 12 months old (or a classic that's still accurate)
- 10–40 minutes to consume
- Directly relevant to the week's topic

**Action tasks must be:**
- Completable in 15–30 minutes
- Produce a tangible output
- Build directly on the resource

**ADHD Notes must be:**
- One sentence max
- Tell the learner WHERE to focus, not just what the topic is
- Acknowledge that the resource might be long/dense

## Code of Conduct

Be kind. We're all learning. No gatekeeping, no elitism.

## Questions?
